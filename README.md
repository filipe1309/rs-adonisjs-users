## :cd: Installation

```sh
git clone git@github.com:filipe1309/rs-adonisjs-users.git
```

```sh
cd rs-adonisjs-users
```

Install dependencies:

```sh
yarn
```

## :runner: Running

```sh
yarn start
```

> Access http://localhost:3333/api/users to see the API
